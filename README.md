## 🎯 **¿Qué es Business Control Panel?**

**Business Control Panel** es un **dashboard administrativo tipo SaaS** diseñado para la gestión empresarial moderna. Una interfaz completa que te permite visualizar métricas clave, gestionar usuarios y analizar el rendimiento de tu negocio en tiempo real.

> 💡 **Ideal para:** Startups, PYMEs, equipos de producto y cualquier negocio que necesite una visión clara de sus operaciones.

## ✨ **Características Principales**

| Característica | Descripción |
|----------------|-------------|
| 📊 **Dashboard Inteligente** | Métricas en tiempo real con indicadores de rendimiento |
| 📈 **Gráficas Interactivas** | Visualización de datos con Recharts (barras, áreas, líneas) |
| 👥 **Gestión de Usuarios** | Tabla completa con filtros, búsqueda y acciones |
| 🌙 **Modo Oscuro/Claro** | Interfaz adaptativa con persistencia en localStorage |
| 📱 **100% Responsive** | Funciona perfectamente en móvil, tablet y desktop |
| 🎨 **Diseño Moderno** | UI limpia con Tailwind CSS y animaciones suaves |
| 🚀 **Alto Rendimiento** | Construido con Vite para cargas ultrarrápidas |
| 🔒 **TypeScript** | Código tipado para mayor seguridad y mantenibilidad |

## 🛠️ **Tecnologías Utilizadas**

### **Frontend Stack**

| Tecnología | Logo | ¿Qué hace? |
|------------|------|------------|
| **React 18** | ⚛️ | Biblioteca para construir interfaces de usuario interactivas y componentizadas |
| **TypeScript** | 📘 | Superset de JavaScript que añade tipado estático para código más seguro y escalable |
| **Tailwind CSS** | 🎨 | Framework CSS utilitario para estilos rápidos y personalizables sin salir del HTML |
| **Vite** | ⚡ | Bundler ultrarrápido con HMR (Hot Module Replacement) para desarrollo ágil |
| **Recharts** | 📊 | Librería de gráficos composables basada en D3 para visualización de datos |
| **Lucide React** | 🎯 | Iconos SVG modernos y consistentes para React |
| **Faker.js** | 🎭 | Generador de datos falsos para prototipado y desarrollo sin backend |

### **Herramientas de Desarrollo**

- **ESLint** - Linter para mantener calidad de código
- **PostCSS** - Procesador CSS para optimización
- **Autoprefixer** - Prefijos CSS automáticos
- **Git** - Control de versiones

## 🚀 **Instalación y Uso**

### **Requisitos Previos**
- Node.js (v18 o superior)
- npm o yarn

### **Pasos para ejecutar el proyecto**

```bash
# 1️⃣ Clonar el repositorio
git clone https://github.com/tu-usuario/business-control-panel.git

# 2️⃣ Entrar al directorio
cd business-control-panel

# 3️⃣ Instalar dependencias
npm install

# 4️⃣ Ejecutar en modo desarrollo
npm run dev

# 5️⃣ Abrir en el navegador
# Ve a http://localhost:5173
